# 🧮 BMI & Calorie Calculator

A modern, interactive **BMI and Calorie Calculator** built with vanilla HTML, CSS & JavaScript. Single file, metric units, fully offline.

🔗 **Live demo:** https://bmi-calorie-calculator-puce.vercel.app/

## Features
- **BMI Calculator** – instant BMI, category & healthy weight range
- **Calorie Calculator** – daily calorie needs (Mifflin–St Jeor) for losing, maintaining & gaining weight
- Clean tabbed UI, metric units, zero dependencies

## Usage
Open `index.html` in any browser — no installation needed.

## Built With
HTML5 · CSS3 · Vanilla JavaScript
